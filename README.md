# 🚨 CAA -- Covid Action Alert

Covid Action Alert (CAA) is a location-based React Native application
that displays the COVID-19 status of establishments directly on Google
Maps.\
It helps users identify safe, moderate, or high-risk locations using
real-time data integration.

------------------------------------------------------------------------

## 📱 Features

-   🗺️ Google Maps Integration\
-   📍 Nearby Search & Place Search (Google Places API)\
-   🔄 REST API Integration\
-   🔥 Firebase Realtime Database & Firestore\
-   💾 Redux Persist (State Persistence)\
-   📊 Live Establishment Status Tracking

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   React Native\
-   Redux + Redux Persist\
-   REST APIs\
-   Google Maps JavaScript API\
-   Google Places API\

## ⚙️ Installation

### 1️⃣ Clone the Repository

    git clone https://github.com/YOUR_USERNAME/caa-mobile-app-development-19073574-react.git
    cd caa-mobile-app-development-19073574-react

### 2️⃣ Install Dependencies

    npm install
    # or
    yarn install

------------------------------------------------------------------------

## 🔐 Environment Setup

Create a `.env` file in the root directory:

    GOOGLE_MAPS_API_KEY=your_google_maps_key
    FIREBASE_API_KEY=your_firebase_key

⚠️ Do NOT commit API keys to the repository.

------------------------------------------------------------------------

## ▶️ Running the Application

### Android

    npx react-native run-android

### iOS

    npx react-native run-ios
